# springboot_tutorial

此demo用户Sap Cloud Platform 中CF环境先部署spring boot程序
    
* 本地：

    命令为： `mvn clean install -P local`
    
* 远程：

    打包  `mvn clean install -P cf`
    
    部署  `cf push`
