# springboot_tutorial

此demo用户Sap Cloud Platform 中CF环境先部署spring boot程序

SAP Cloud platform 部署CF应用的步骤为：

- CF需要指向需要部署的endpoint

    命令为： `cf api XXXXX.XXXX`
    
- cf 登陆

    命令为： `cf login`
    
- cf 推送
    
    命令为： `cf push`
